- HTML was first proposed by Tim Berners Lee in 1989 while working at CERN. - The first publicly available description of HTML was published in 1991. HTML VERSIONS * HTML 1.0(1991) - Basic Text Formatting and hyperlinks. * HTML 2.0(1995) - Tables,Forms and client side scripting. * HTML 3.2(1997) - Style sheets,scripts and improved support for tables and forms. * HTML 4.0.1(1999) - Improved support for style sheets and deprecated some elements. * HTML 1.0(2000) - A reformulation of HTML 4.0.1 using XML syntax. * HTML 5(2014) - The current version of HTML, which introduces new features such as videos,audios,canvas and improved semantic elements.
